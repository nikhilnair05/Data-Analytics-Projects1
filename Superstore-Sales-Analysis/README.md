# Superstore Sales Analysis

## Overview
Analysis of 9,994 retail orders to identify profitability drivers and loss-making products.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- MySQL (CTEs, Window Functions)
- Power BI (DAX, Power Query)

## Key Findings
- Tables and Bookcases generate high sales but lose money on every order
- Heavy discounting (>20%) costs $135K in annual profit
- West region has highest profit margin at 14.94%
- Central-Furniture combination has worst margin at -1.75%

## Files
- `week2_project.py` — Pandas EDA and analysis
- `week3_dashboard.py` — Matplotlib/Seaborn visualizations
- `week4_project.sql` — SQL business analysis queries
- `Superstore_Dashboard.pdf` — Power BI dashboard export
